BrightView — LED screens static site scaffold

This repository contains a simple static website scaffold for a small LED screen company. It includes:
- `index.html` — homepage with featured products
- `products.html`, `gallery.html`, `about.html`, `contact.html`
- `css/styles.css` and `js/app.js` for styles and small interactive behaviors

Quick start (local):
1. Open `index.html` in a browser (no build needed).

Deploy to GitHub Pages:
1. Create a repository on GitHub and push the files to the `main` branch.
2. In the repository Settings → Pages, set the source to the `main` branch and the root folder.
